About Sharp Portfolio

This is a web implementation of Harry Markowitz's Portfolio Theory. The tool fetches historical stock data from Yahoo! Finance and uses satistical methods of mean and variance to calculate the most optimal portfolio distribution.
The application is built with Flask framework for the backend and uses basic HTML & CSS for the front end.

What's with the name?

The algorithm behind this application generates 10000 different portfolio distributions and then picks the one from the efficient frontier which has the maximum Sharpe Ratio - a performance measure for investments developed by William F. Sharpe.
Hence the name Sharp Portfolio!
